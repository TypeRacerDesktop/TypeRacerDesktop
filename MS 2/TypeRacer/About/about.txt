   This application has been developed by Georgi Kratchkov and Hristo Georgiev, in order to improve user's keyboard input speed. The user can choose between 3 levels of difficulty (150, 250, 350) which respectivly represents the text length in words count. In order to make it harder and interesting, the application has many different texts built-in. It has been developed under 'java'.
